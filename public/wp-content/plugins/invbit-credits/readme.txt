Generate a credits page with a customizable shortcode.

== Description ==

WordPress plugin to generate a "Credits" page via shortcode with two columns:
* Left: Custom content from admin panel (title, description, features)
* Right: Fixed company information

== Installation ==

1. Upload plugin files to `/wp-content/plugins/invbit-credits/`
2. Activate plugin in WordPress
3. Go to 'Invbit Credits' menu to configure
4. Click 'Create Credits Page'
